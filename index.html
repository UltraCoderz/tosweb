1. Boot Sector Virus: Infects the boot sector of storage devices, making them unable to boot properly.

2. File Infector Virus: Infects executable files, spreading when the infected file is executed.

3. Macro Virus: Infects documents or templates that use macros, such as Word or Excel documents.

4. Polymorphic Virus: Changes its code to evade detection by antivirus software.

5. Metamorphic Virus: Completely rewrites its code with each infection to avoid detection

1.Multipartite Virus: Infects both boot sectors and executable files.

2.Resident Virus: Lodges itself in the computer's memory and can infect files as they are opened or created.

3.Non-Resident Virus: Does not stay in the computer's memory and typically infects files when they are executed.

4.Direct Action Virus: Attaches itself to specific files and activates when those files are opened or executed.

5.Overwrite Virus: Overwrites files with its own code, rendering them unusable.

1.Cluster Virus: Infects entire directories or folders by infecting the directory's index table.

2.File System Virus: Attacks the file system or specific files within it.

3.Stealth Virus: Tries to evade detection by antivirus software by actively hiding itself.

4.Sparse Infectors: Infects files sporadically to avoid detection.

5.Camouflage Virus: Mimics legitimate programs to avoid detection.

2.FAT Virus: Infects the File Allocation Table (FAT) of the disk.

1.Rootkit: Conceals malicious software from the user and antivirus programs by altering system functionality.

2.Logic Bomb: Activates when specific conditions are met, such as a certain date or action taken by the user.

3..Botnet: Uses infected computers to perform coordinated tasks, often without the user's knowledge.
